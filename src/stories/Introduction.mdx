# Silvertree Design System

Welcome to the Storybook for the Silvertree Design System.

This is the **governance layer** — every component here is built directly from
the design tokens defined in `src/tokens/tokens.css`. If it lives here, it is
the approved, correct version.

## What's here

- **Tokens** — the source of truth for colour, typography, spacing, and more
- **Components** — React UI components built from those tokens
- **Stories** — visual tests showing every state and variant of each component

## How to use

Browse the sidebar on the left. Each component shows all its variants and allows
you to interact with its properties in the **Controls** panel below the preview.
